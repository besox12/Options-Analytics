About & Methodology
Overview

This tool is a standalone HTML/JS derivative analytics workbench — zero dependencies, no server required. It implements Black-Scholes-Merton option pricing, analytical Greeks, Monte Carlo simulation, and various risk decomposition techniques. Built for educational use and interview preparation in energy/commodity trading.

Black-Scholes-Merton Framework

Option prices are computed under the risk-neutral measure using the BSM closed-form solution:

C = S₀ N(d₁) − K e−rT N(d₂)
P = K e−rT N(−d₂) − S₀ N(−d₁)
d₁ = [ln(S/K) + (r + σ²/2)T] / (σ√T)    d₂ = d₁ − σ√T

The cumulative normal distribution N(·) is approximated using the Abramowitz & Stegun rational polynomial (error < 1.5×10⁻⁷). Standard assumptions: log-normal returns, constant volatility, no dividends, European exercise, frictionless markets.

Greeks — Analytical Sensitivities

All Greeks are computed analytically from the BSM closed-form derivatives. For portfolios, individual Greeks are summed (linearity of differentiation).

Delta: ∂V/∂S = N(d₁) [call], N(d₁)−1 [put]
Gamma: ∂²V/∂S² = φ(d₁) / (Sσ√T)
Vega: ∂V/∂σ = Sφ(d₁)√T
Theta: ∂V/∂t = −Sφ(d₁)σ/(2√T) − rKe−rTN(d₂) [call]
Rho: ∂V/∂r = KTe−rTN(d₂) [call]

where φ(·) is the standard normal PDF. Note: Vega and Gamma are identical for calls and puts (put-call parity).

Monte Carlo Simulation

GBM paths are generated under the risk-neutral measure:

S(t+dt) = S(t) · exp[(r − σ²/2)dt + σ√dt · Z],   Z ~ N(0,1)

Normal variates are generated using the Box-Muller transform (exact, not approximate). The fan chart displays P5/P25/P50/P75/P95 percentile bands computed cross-sectionally at each time step. Terminal values are used for P&L distribution and risk metrics.

Risk Metrics

VaR (Value at Risk): Historical simulation from MC terminal distribution. α-quantile of P&L distribution.
CVaR (Expected Shortfall): Mean of losses beyond VaR threshold — coherent risk measure.
Sharpe / Sortino / Calmar: Computed from MC path returns. Sortino uses downside deviation only.
Jarque-Bera: Tests normality of terminal returns using skewness and kurtosis moments.

Exotics Lab — Path-Dependent Pricing

Path-dependent options (Asian, Lookback, Barrier) are priced via Monte Carlo since no closed-form BSM solution exists. Greeks are computed via bump-and-reprice (finite difference on MC price):

Δ ≈ [V(S+h) − V(S−h)] / 2h
Γ ≈ [V(S+h) − 2V(S) + V(S−h)] / h²
ν ≈ [V(σ+δσ) − V(σ−δσ)] / 2δσ

Each Greek requires 2-3 independent MC runs, so standard errors compound. Increase path count for precision.

Volatility Smile

The parametric smile model is a second-order polynomial in moneyness:

σ(K) = σ_ATM + skew · (K−S)/S + convexity · ((K−S)/S)²

This captures the first-order skew (negative for equity/power, positive for some commodities) and the convexity (smile curvature). When applied, all BS-based pricing uses the strike-dependent volatility.

Implied Volatility Solver

IV is extracted using Newton-Raphson iteration on the BSM pricing function:

σ_{n+1} = σ_n − [C_BS(σ_n) − C_mkt] / Vega(σ_n)

Convergence is typically achieved in 5-10 iterations (tol = 10⁻⁶). The Put-Call Parity checker verifies C − P = S − Ke−rT and flags potential arbitrage when the residual exceeds 0.50.

Hedge Advisor

The hedge advisor solves a linear system to neutralize target Greeks. For Δ+Γ+ν neutralization:

[Γ₁ Γ₂] [n₁] [−Γ_port]
[ν₁ ν₂] [n₂] = [−ν_port]

n_fut = −(Δ_port + n₁Δ₁ + n₂Δ₂)

Two calls with different K/T provide independent Γ and ν exposure. Futures are used for residual delta because they have Δ=1 and zero Γ/ν.

P&L Taylor Decomposition

The P&L is decomposed via a second-order Taylor expansion of the BS price:

ΔP&L ≈ Δ·δS + ½Γ·(δS)² + Θ·δt + ν·δσ + residual

This shows whether P&L was driven by directional movement (Δ), convexity (Γ), time decay (Θ), or volatility changes (ν). The residual captures higher-order and cross-effects.
